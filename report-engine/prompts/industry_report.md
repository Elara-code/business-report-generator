# 行业商业模式分析报告 Prompt

用户输入主题：**{subject}**（一个行业或领域，例如"咖啡"、"新能源汽车"、"SaaS"）
报告类型：`industry`（行业商业模式分析）

## 你的任务

生成一份**{subject}**行业的商业模式分析报告。读者是有投资意向的创业者和投资人，他们需要：
- 快速理解行业全景
- 看到具体可量化的市场数据
- 识别竞争格局和潜在机会
- 形成投资 / 创业的判断依据

## 数据可视化与标签约束（重要！）

为保证 SVG 图表渲染时不溢出 / 重叠，**所有图表的文本标签必须遵守以下长度限制**：
- `categories`（条形图/矩阵图点位名）：**≤ 10 个字符**（中文按 1 字算）
- `axes`（雷达图维度名）：**≤ 6 个字符**
- 折线图 X 轴时间标签：年份/季度即可（≤ 8 字）
- 画布 9 宫格条目：**每条 ≤ 12 字**，超过请用简称或缩写
- 矩阵图点位名（product/player）：**≤ 8 字**

如果数据本身标签过长（如"中国联合网络通信集团有限公司"），请在 JSON 里用**简称 + 完整名放附录**的方式处理。

## 各章节写作要求

### 1. 执行摘要
- 150-250 字
- 结论先行：行业当前处于什么阶段、增速如何、谁在主导、机会在哪
- 用 1-2 个数字佐证（如市场规模、CAGR）
- 末尾给一句"机会窗口 / 风险提示"判断

### 2. 市场概览
- **必须配 bar/line 图**：最近 3-5 年市场规模（亿元/美元）+ CAGR
- 数据格式：`{"categories": ["2020","2021","2022","2023","2024"], "values": [...], "unit": "亿元"}`
- 正文补充：渗透率、用户规模、行业生命周期阶段
- 引用至少 2 个权威数据源（艾瑞、易观、Statista 等）

### 3. 价值链分析
- **必须配 value_chain 图**
- 数据格式：`{"stages": [{"name": "上游原材料", "items": ["豆种", "烘焙厂"], "margin": "高"}, ...]}`
- 标出每个环节的毛利率、关键企业、议价能力

### 4. 商业模式画布
- **必须配 canvas 图**
- 数据格式：
```
{
  "key_partners": ["..."],
  "key_activities": ["..."],
  "value_propositions": ["..."],
  "customer_relationships": ["..."],
  "customer_segments": ["..."],
  "key_resources": ["..."],
  "channels": ["..."],
  "cost_structure": ["..."],
  "revenue_streams": ["..."]
}
```
- 9 个格子每格 3-5 个要点

### 5. 竞争格局
- **必须配 radar 图**：波特五力评分 0-5
- 数据格式：`{"axes": ["供应商议价", "购买者议价", "新进入者", "替代品", "行业内竞争"], "values": [3.5, 4.0, 2.5, 3.0, 4.5]}`
- 正文：高度集中 / 分散？头部公司份额？竞争维度（价格 / 品牌 / 渠道 / 技术）？

### 6. 关键玩家
- **必须配 bar 图**：Top 5 玩家市占率
- 数据格式：`{"categories": ["公司 A", "公司 B", ...], "values": [25, 18, 12, 8, 5], "unit": "%"}`
- 正文：每家一句话点评（定位 / 优势 / 弱点）

### 7. 趋势与机会
- 3-5 个趋势，每条配一个数据点 / 案例
- 包含：技术、用户、政策、资本四个视角

### 8. 风险与建议
- 3-5 条风险，按"发生概率 × 影响"排
- 末尾：给创业者 3 条具体建议
